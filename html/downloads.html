Rabbit Hole Main Menu

The Rabbit Hole Main Menu project was a personal project that I worked on in my spare time. I experienced some menus from a few indie games such as Ori and the Blind Forest and Hollow knight and thought to myself: “I wonder how easy it would be to make my own main menu for any games I make in the future?” And with that question, I set out on a journey to making a menu for a game I called Rabbit Hole. The goal of the project was to learn the ins and outs of how to make a solid main menu from the ground up.

Deep Stone Crypt: Crypt Security Recreation in Unity

Deep Stone Crypt: Crypt Security Recreation is a personal project that I took on in Unity. I play a game called Destiny 2 that is developed by Bungie. Within the game, there is an activity called a raid. Each raid is split up into a set of encounters where you are given a challenge or puzzle to solve. This encounter is one that I personally like a lot because of all the different ways it can be solved. Because of my deep fascination with the encounter design, I decided that I wanted to recreate their mechanics from the ground up. The goal of the project was to see how well I truly understood the rules of encounter and to code those rules myself.

Futuristic UI

Futuristic UI is a personal project that I created with HTML, CSS, and JavaScript. The project was my first attempt at creating a flat style menu using tile-based flow. I learned about a modern approach to main menus called tile-based flow (found in games like Hitman). I also learned the importance of color and font selection for a menu. With all that being said, I wanted to understand how these game UI concepts would work for a website instead of a game’s menu system. The goal of the project was to replicate this UI concept in a web browser and see how closely I could get the website to look like a production game menu.
